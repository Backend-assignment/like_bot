# Setup environment

## Create virtual environment

```bash
python3 -m venv venv
```

## Activate virtual environment

```bash
source venv/bin/activate
```

## Install requirements

```bash
pip install -r requirements.txt
```

## Git ignore

```bash
echo "venv" >> .gitignore
```

# Set environment variables

## Telegram bot token

```bash
export TOKEN="your token"
```
